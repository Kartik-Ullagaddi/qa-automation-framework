# QA Automation Framework

## Overview
This project is a Selenium-based automation framework built using Java and TestNG following Page Object Model (POM).

## Features
- Selenium WebDriver
- TestNG Framework
- Page Object Model (POM)
- Data-driven testing using JSON
- Config management using properties file
- Positive and negative test scenarios

## Tech Stack
- Java
- Selenium
- TestNG
- Maven

## Test Scenarios
- Valid login test
- Locked user negative test

## How to Run
Run testng.xml or use:
mvn clean test
